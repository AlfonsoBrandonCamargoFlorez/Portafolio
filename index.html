HTML

HTML (Lenguaje de Marcado de Hipertexto) es el estándar para crear y estructurar contenido en la web. Utiliza un sistema de etiquetas para definir elementos como encabezados, párrafos, enlaces e imágenes, permitiendo la construcción de páginas web.

CSS

CSS (Hojas de Estilo en Cascada) es un lenguaje de hojas de estilo que me permite controlar la presentación y el diseño visual de los elementos HTML en las páginas web. Mejora la estética y el diseño de mis proyectos, permitiendo personalizar colores, fuentes y layouts.

JavaScript

Vuelos

Descubre mi innovadora solución para la gestión de vuelos, diseñada para transformar la administración en el cielo. He creado un sistema integral que abarca la gestión de aviones, aeropuertos, pasajeros y escalas. Situación: Me enfrenté al desafío de optimizar la coordinación y administración de múltiples aspectos en la operación de vuelos. Tarea: Desarrollar una plataforma completa que permita gestionar de manera eficiente cada componente del proceso de vuelo. Acción: Diseñé e implementé un sistema versátil y adaptable que ofrece funcionalidades avanzadas para la creación, modificación y búsqueda de datos, ajustándose a los roles y permisos específicos. Resultado: Obtengo una solución que asegura una operación de vuelos fluida y eficaz, optimizando la administración y mejorando la experiencia general.

Gestion Universidad

Explora mi innovadora solución para la gestión universitaria, diseñada para optimizar la administración académica. Este programa permite gestionar salones, profesores y la cantidad de estudiantes por salón, asegurando que no haya conflictos en los horarios al seleccionar materias. Situación: Necesitaba una herramienta que facilitara la organización y planificación de los recursos académicos. Tarea: Crear un sistema eficiente para manejar la asignación de salones, la programación de profesores y la distribución de estudiantes. Acción: Desarrollé una plataforma que gestiona la disponibilidad de salones, asigna profesores y controla los horarios de manera que se eviten solapamientos para los estudiantes. Resultado: Conseguí una solución que mejora la organización académica, evitando conflictos de horario y optimizando la gestión de recursos.
