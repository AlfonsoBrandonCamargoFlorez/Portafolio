HTML

HTML (Lenguaje de Marcado de Hipertexto) es el estándar para crear y estructurar contenido en la web. Utiliza un sistema de etiquetas para definir elementos como encabezados, párrafos, enlaces e imágenes, permitiendo la construcción de páginas web.

Aprende más

CSS

CSS (Hojas de Estilo en Cascada) es un lenguaje de hojas de estilo que me permite controlar la presentación y el diseño visual de los elementos HTML en las páginas web. Mejora la estética y el diseño de mis proyectos, permitiendo personalizar colores, fuentes y layouts.

JavaScript

JavaScript es un lenguaje de programación que permite añadir interactividad y funcionalidades dinámicas a las páginas web. Esencial para crear experiencias web ricas, permite manipular elementos en tiempo real y responder a las acciones del usuario.

Spring Boot

Spring Boot es un framework basado en Java con el que puedo construir aplicaciones robustas de manera eficiente, aprovechando su soporte para microservicios, seguridad y bases de datos, lo que optimiza el proceso de desarrollo y despliegue.
